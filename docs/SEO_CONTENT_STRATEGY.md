## SEO Optimization Checklist for Each Article

### On-Page SEO
- [ ] Primary keyword in title (front-loaded)
- [ ] Primary keyword in first 100 words
- [ ] Secondary keywords naturally distributed (2-3 per 1,000 words)
- [ ] Meta description (150-160 chars) with primary keyword
- [ ] H1 tag (only one per page)
- [ ] H2/H3 hierarchy (proper structure)
- [ ] Internal links (3-5 to related articles)
- [ ] External links (2-3 to authoritative sources: Parole Board, Justice.gc.ca)
- [ ] Image alt text (descriptive, includes keywords where natural)
- [ ] Word count: 1,500-3,000 words (minimum)

### Technical SEO
- [ ] URL slug: descriptive, lowercase, hyphens (e.g., `/blog/record-suspension-eligibility`)
- [ ] Schema markup: Article schema with author, date, word count
- [ ] Mobile responsive
- [ ] Page load speed < 3 seconds
- [ ] No broken internal links
- [ ] Canonical tag (if applicable)

### User Experience
- [ ] Clear CTA (e.g., "Check Your Eligibility" button)
- [ ] Scannable format (bullet points, short paragraphs)
- [ ] Related articles sidebar
- [ ] FAQ section (schema markup)
- [ ] Table of contents (for 2,000+ word articles)

### Content Quality
- [ ] Fact-checked against Parole Board of Canada resources
- [ ] Legal accuracy verified
- [ ] Canadian-specific information
- [ ] Updated publication date
- [ ] Author bio/credentials
- [ ] Sources cited

## Long-Tail Keyword Strategy

### Why Long-Tail Keywords?

**Advantages for PardonPath:**
1. **Lower competition** — Easier to rank for specific queries
2. **Higher intent** — Users know what they want
3. **Better conversion** — Specific problems = specific solutions
4. **Faster ranking** — Newer sites can rank in months, not years
5. **Less paid competition** — Lower CPC if you ever run ads

**Example:**
- ❌ "record suspension" (1,000+ competitors, $5+ CPC)
- ✅ "record suspension for drug offence Ontario" (50 competitors, $0.50 CPC, 95% intent match)
